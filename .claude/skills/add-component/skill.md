# Add shadcn Component

Add a shadcn/ui component to the Carelog web app.

Steps:
1. Run: `cd apps/web && pnpm dlx shadcn@latest add {args}`
2. Verify: `ls apps/web/components/ui/{args}.tsx` — confirm file exists
3. Report: "✓ {args} installed at apps/web/components/ui/{args}.tsx" or show the error output if it failed.

Usage: `/add-component button`, `/add-component sheet`, `/add-component data-table`
