# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## CLAUDE.md Enhancement Trigger

After completing any substantial work in this project (multi-step task, feature, bug fix spanning >1 file, or any session-meaningful effort), run a brief retrospective and propose CLAUDE.md updates. Cover:

1. **Goal** — what was the user actually trying to accomplish
2. **Tasks required** — the steps the work decomposed into
3. **Issues encountered** — what went wrong, what surprised, what required correction
4. **Resolution** — how each issue was handled
5. **What was executed vs planned** — explicit drift explanation: what got added, dropped, or reordered relative to the original ask, and why

Then propose specific CLAUDE.md edits (new rules, refined rules, removed rules) that would have prevented the drift or sped up the work next time. Do not edit CLAUDE.md without explicit approval — surface proposals only.
