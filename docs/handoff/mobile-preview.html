What changed on small screens

Every page reflows under 720 px. The sidebar dissolves into a fixed bottom tab bar in Deep Plum Ink; primary actions take the head; secondary actions slide into a More sheet.

  • NavFive tab bar: Today, Journal, Shifts, Meds, More — Team, Documents, Brief, Settings live in the More sheet.
  • TodayHero stays editorial; cards stack; pattern footer slips below Coming up.
  • JournalQuick-log composer is full-width; filter chips scroll horizontally; entry actions floats absolute.
  • ShiftsThe weekly calendar is hidden and replaced by an agenda list grouped by day.
  • MedsThe table becomes labeled cards — name + dose, schedule, supply, last-taken stacked.
  • DocumentsFolder grid stays 2-up; row metadata simplifies to name + folder.
  • SettingsVertical tab nav becomes a horizontal pill scroller; fields stack label-over-input.
  • BriefEditorial Fraunces holds up at narrow widths; body type rescales.
